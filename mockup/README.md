# BI-SU 会員アプリ HTMLモックアップ

Flutter本実装前の「デザインの設計図」となる動くモックアップ。
表示されている会員情報・注文・距離はすべてダミー（商品名・価格・店舗情報・ランク制度は公式サイト掲載情報を参照）。

## ファイル構成

| パス | 役割 |
|---|---|
| `bisu-app-mockup.html` | **配布用の単一ファイル版**（CSS/JS/画像すべて埋め込み・オフライン動作） |
| `src/` | 開発用ソース（index.html / styles.css / scenes.js / app.js / assets/） |
| `build-singlefile.ps1` | src/ から単一ファイル版を再生成するスクリプト |
| `reference/` | 公式サイトから取得した参照資料（ランク表画像・元画像） |

## 修正の流れ

1. `src/` 内のファイルを編集
2. プレビュー: launch.json の `bisu-app-mockup`（port 8094, src/ を配信）
3. `powershell -ExecutionPolicy Bypass -File build-singlefile.ps1` で配布用を再生成

## 主な仕様

- 会員ランク5段階: MEMBER / SILVER(年間30万円〜) / GOLD(50万〜) / PLATINUM(100万〜) / DIAMOND(200万〜)
  - ポイント換算率 1/2/3/5/10pt（公式クラブメンバーズプログラム準拠）
  - 公式ランクカラー: PLATINUM=青系・DIAMOND=深紅系（公式サイトのランク表画像より）
- 起動演出: ランク別に「上空→島→ジャングル→洞窟入口→洞窟内→巣」の2シーン連結ズーム（SVGアート）
- **MEMBER・SILVER・GOLDはAI生成動画版の起動演出**: `journey-member.mp4`（720×1280・使用区間0〜3.05s）/ `journey-silver.mp4`（360×640・4s全編）/ `journey-gold.mp4`（360×640・4.04s全編）。最終フレームをキャプチャしてホーム背景に引き継ぐ。`RANKS[].video`（+必要なら`videoEnd`）に動画パスを足せば他ランクも動画化できる（動画なしのランクは従来のSVG演出）
- 開発サーバーは `serve.py`（キャッシュ無効化付き）。編集→リロードで必ず最新が反映される
- **ホームのエンブレムは3D**（Three.js）: 公式ロゴマークを手作業ベクター化→押し出し3D化。ランクカラーの金属質感。マークは静止させ、ライティングの角度を動かして光を流すスイープ演出（回転・浮遊なし）。WebGL不可環境はフラットSVGに自動フォールバック。形状パスは `logo3d.js` の `SWALLOW_D`（公式AIデータ入手後に差し替え推奨）
- ホーム背景=演出の着地シーン。設定画面のランク切替で5パターンをリプレイ確認可能
- 背景はSVGレイヤー構造のため、実写真への差し替えが容易
- 動画の正本は `src/assets/journey-member.mp4`（yukaさん生成・720×1280・10秒。2026-06-12にプロジェクト直下から移動・リネームして1本化）。差し替え時はこのファイルを上書き→再ビルド
