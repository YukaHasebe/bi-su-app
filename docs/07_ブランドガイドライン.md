## 1. ブランドアイデンティティ

### 1.1 ブランドコンセプト

> ツバメの巣の可能性を、自然への敬意と科学的根拠で世界に届ける。

BI-SU は単なる商品ブランドではない。人の未来を変え、動物の未来を守り、自然との共生を証明し、天然物の可能性を科学で世界に示すために存在するブランドである。

アプリはこの世界観を日常のタッチポイントとして会員に届ける役割を担う。

### 1.2 ブランドの核となる価値観

| 価値観 | 説明 |
|---|---|
| 何も傷つけない | 人、動物、自然、アナツバメを傷つけないブランド |
| 天然物の可能性を科学で証明する | 感覚ではなくエビデンスで世界に届ける |
| 世界基準を作る | 品質・技術・研究・ブランド表現まで含めた世界基準 |
| 人と動物の未来をつなぐ | BI-SU は人へ、MIRANEST は動物へ。根本は同じ |

### 1.3 BI-SU と MIRANEST のブランド区別

BI-SU と MIRANEST は同じ思想を根幹に持つが、ブランドとしてのトーン&マナーは明確に異なる。本ガイドラインは **BI-SU（人のための高級ツバメの巣ブランド）** に特化する。

| 観点 | BI-SU | MIRANEST |
|---|---|---|
| 対象 | 人（スキンケア・インナービューティー） | ペット（愛犬・愛猫の健康） |
| トーン | 上品・ラグジュアリー・温もりのある高級感 | モダン・スタイリッシュ |
| 基調色 | ウォームアイボリー × シャンパンゴールド | ライトグレー × ミディアムグレー |
| 書体の方向性 | 明朝体ベース（格式・品格） | ゴシック体ベース（モダン・クリーン） |
| 質感 | 温かみのある素材感・絹のような上質さ | クール・ミニマル |

> BI-SUアプリの制作では、`vault/Brand/基調色ガイド.md`（MIRANESTの色定義）を参照しないこと。本ガイドラインのカラーパレット（セクション2）を使用する。

### 1.4 ボイス&トーン

**基本トーン**: 上品 / 力強い / 神秘的 / 科学的 / 感動的 / 未来志向 / 誠実 / 自然への敬意

| 方針 | 具体例 |
|---|---|
| 世界観がある文章にする | 読んだ人が「これはただの商品ではない」と感じる表現 |
| 上品・ラグジュアリー寄り | 百貨店コスメブランドの格式。高級感と温かみを両立 |
| 安っぽい煽りを避ける | 激安、バズ狙いの軽い言葉、過剰な売り込みはNG |
| 作り手スタンスで語る | 「最上級」ではなく「こだわり」。比較・優位性の断定ではなく姿勢 |
| 各セクション自己完結 | 文が見出しをまたいで完成する構造は避ける（通販的煽りに見える） |
| 抽象1:具体3 | リードに世界観、本文に事実・数字を織り交ぜる |

**好ましい言葉**:
無限の可能性 / 生命の根源 / 天然の叡智 / 科学で証明する / 世界基準 / 未来を変える / 本物だけが持つ力 / 自然と科学の融合 / 美しさのその先へ / 健康のその先へ

**避けるべき言葉**:
激安 / バズ狙いの軽い言葉 / 根拠のない断定 / 医薬品的効能表現 / 他社を下げる表現 / 自然や動物への敬意がない表現

## 2. カラーパレット

### 2.1 BI-SU ブランドカラーの方針

BI-SUのブランドカラーは **ウォームアイボリー × シャンパンゴールド**。公式サイト（bi-su.jp）およびLP全体で一貫して使用されている。

| 要素 | 方向性 |
|---|---|
| 背景 | 温かみのあるアイボリー〜オフホワイトのグラデーション |
| ブランドアクセント | シャンパンゴールド（`#C6B484` 系統） |
| テキスト | 黒（`#000`）を基本 / ゴールドをナビゲーション・装飾に使用 |
| ダークセクション | ニアブラック（`#1C1C1C`）にゴールドテキスト・装飾を配置 |
| 質感 | 絹のような柔らかさ、温もり、素材感 |

> **MIRANESTとの違い**: MIRANESTのブランド基調色（ライトグレー `#d8d8d8` + ミディアムグレー `#565656`〜`#666666`）はBI-SUには適用しない。BI-SUは温かみのある暖色系が基調。

| シャンパンゴールドはブランドの象徴色 | ナビゲーション・ボタン・アクセント・チップに使う。BI-SUにおいてゴールドは「控えめなアクセント」ではなく中心的な色 |

| 冷たいグレー単色を避ける | グレーを使う場合もウォームグレー寄りに。MIRANESTの `#d8d8d8` 基調にしない |
| ボタニカル柄はAI生成しない | 自社素材をデザイナーが入れる。AI画像生成で作成しない |
| 既存制作物を目視確認してから色を決める | design-brief等の色記述を鵜呑みにしない |

## 3. タイポグラフィ

### 3.1 フォントファミリー

BI-SUは **明朝体（セリフ）を基調とした格式あるタイポグラフィ** を採用する。公式サイト（bi-su.jp）の本文フォントは游明朝であり、アプリもこの方向性を踏襲する。

| 用途 | フォント | フォールバック | 公式サイト対応 |
|---|---|---|---|
| ブランドロゴ・英字装飾 | **Trajan Pro** | Georgia / serif | `--font-trajan` |
| 見出し・ランク名・金額（明朝系） | **Noto Serif JP** | Yu Mincho / Hiragino Mincho ProN | `--font-mincho`（游明朝） |
| 本文・UI要素（ゴシック系） | **Noto Sans JP** | Yu Gothic Medium / Hiragino Kaku Gothic ProN | `--font-gothic`（游ゴシック体） |

Flutter本実装では **Noto Serif JP / Noto Sans JP をアプリに同梱** し、端末差異を排除する。Trajan Pro はロゴ画像として扱い、テキストフォントとしての同梱は不要。

> **MIRANESTとの違い**: MIRANESTはゴシック体（sans-serif）が基調で、明朝体はアクセント的使用。BI-SUは逆に明朝体が主役で、ゴシック体はUIの可読性確保に使う。

### 3.2 フォント使い分けの方針

| 場面 | フォント | 理由 |
|---|---|---|
| 画面タイトル・セクション見出し | 明朝系 | BI-SUの格式・高級感を表現 |
| ランク英字（MEMBER等） | 明朝系 | 公式サイトの Trajan Pro 的な品格を踏襲 |
| 金額表示 | 明朝系 | 高級感・信頼感 |
| ブランドストーリー本文 | 明朝系 | 読み物としての品格 |
| UIラベル・ボタン・入力欄 | ゴシック系 | 可読性・操作性 |
| 補助テキスト・注記 | ゴシック系 | 情報の階層を明確化 |
| リスト・表形式データ | ゴシック系 | 一覧性 |

### 3.3 文字サイズ・ウェイト

| 要素 | サイズ（参考値） | ウェイト | フォント | 備考 |
|---|---|---|---|---|
| ランク英字（ホーム） | 40px | Regular〜Medium | 明朝系 | letter-spacing 0.3em 前後の疎組み |
| 画面タイトル | 20〜24px | Medium〜Bold | 明朝系 | — |
| セクション見出し | 16〜18px | Medium | 明朝系 or ゴシック系 | コンテンツ性の高いセクションは明朝 |
| 本文 | 14〜16px | Regular | ゴシック系 | — |
| 補助テキスト | 12〜13px | Regular | ゴシック系 | ink-soft 色を使用 |
| 金額表示 | 16〜20px | Medium | 明朝系 | 税込表記を付す |
| `MEMBERSHIP STATUS` ラベル | 小さめ | Regular | ゴシック系 | 英字大文字・letter-spacing広め |
| 日本語ステータス名 | 中程度 | Regular | ゴシック系 | ランク英字の下に配置 |

### 3.4 タイポグラフィルール

| ルール | 詳細 |
|---|---|
| 英字ランク名は疎組み | letter-spacing 0.3em 前後で上品な間隔を確保 |
| 商品名は最大2行 + ellipsis | 長い商品名（例「BI-SUフェイスマスク Type-R（4枚入り）」）の折返し制御 |
| 会員氏名は外字・長尺を考慮 | 15文字以上の氏名への対応 |
| 多言語対応時は英語の文字幅も考慮 | 日本語/English切替（v2.1） |
| 税込表記を明示 | 金額表示には「（税込）」等を付す |

## 4. ランクシステム

### 4.1 ランク階層

| ランク | 年間購入金額 | ポイント換算率 | バースデーギフト | シーズナルギフト | イベントご招待 |
|---|---|---|---|---|---|
| MEMBER | — | 100円 = 1pt | 100pt | - | - |
| SILVER | 30万円〜 | 100円 = 2pt | ○ | - | - |
| GOLD | 50万円〜 | 100円 = 3pt | ○ | ○ | ○ |
| PLATINUM | 100万円〜 | 100円 = 5pt | ○ | ○ | ○ |
| DIAMOND | 200万円〜 | 100円 = 10pt | ○ | ○ | ○ |

> 正式な会員制度名称: **CLUB MEMBERS PROGRAM**

### 4.2 ランク判定ルール

| 項目 | ルール |
|---|---|
| 表示ランク | 前年（N-1年）1/1〜12/31の購入累計で確定した当年（N年）の公式ステータス |
| 進捗バー | 当年（N年）1/1からの購入累計 / 次ランク閾値 → 翌年（N+1年）ステータスの見込み |
| 進捗0%の扱い | 当年実績が0円でも進捗バーは0%で表示（非表示にしない） |
| ステータス再判定 | 毎年再判定（下位へ戻ることもある） |
| 演出の出し分け | 現ステータス準拠 |
| DIAMOND到達時 | 進捗100%固定 + お礼メッセージ |

### 4.3 ランクバッジ・表示ルール

| 要素 | 表示仕様 |
|---|---|
| 3Dロゴエンブレム | ランクのメインカラー（ツバメ）+ サブカラー（円環）。ゆらぎ回転・浮遊・後光エフェクト付き |
| ランク英字 | 明朝系・40px・疎組み（letter-spacing 0.3em）。ホーム画面中央に配置 |
| 日本語ステータス名 | ランク英字の下に表示 |
| ロケーション表記 | 演出テーマに基づく英字 + 日本語（例: `RAINFOREST -- 熱帯雨林の奥へ`） |
| 進捗バー | ランクのメインカラーで塗りつぶし。「{次ランク} まであと ¥xxx」を表示 |
| 還元率チップ | ランクのメインカラー背景。「100円 = Npt」形式 |
| アバター（設定画面） | ランクのメインカラー背景にイニシャル表示 |
| 仮会員証 | MEMBER色を適用。「仮会員証」ラベルを表示 |

## 6. 禁止事項

### 6.1 ブランド表記に関する禁止事項

| カテゴリ | NG | 正しい表現 |
|---|---|---|
| BI-SUの表記 | BI-SU（美巣） | BI-SU |
| 会員の呼称 | 会員様 / メンバー | 会員 |
| 会員の呼称（英語） | — | Member / Gold / Platinum（英語の頭文字大文字はOK） |
| 商品の呼称 | 製品 / プロダクト / 商品（単体） | BI-SU商品 / MIRANEST商品 / 具体的な商品名 / 日々のケア |
| 制度名 | Members Club / Member's Club / メンバーズクラブ | CLUB MEMBERS PROGRAM |
| アンバサダー | 小雪をアンバサダーとして記載 | 記載しない（契約終了済み） |

### 6.2 医薬品的表現の禁止

| NG表現 | 理由 |
|---|---|
| 治る / 効く / 改善する / 予防する | 薬機法違反のおそれ |
| 飲めば治る / 必ず効く | 断定的な効能表現 |
| 免疫力アップ / 細胞活性化 | 医薬品的な効能表現に該当するおそれ |
| 疾病の治療・改善を示唆する表現 | 健康食品・化粧品では使用不可 |

外部発信（LP・SNS・広告・メルマガ・プレスリリース）では、薬機法・景品表示法・獣医療広告の確認を通した表現のみ使用する。

### 6.3 トーン・表現の禁止事項

| NG | 理由 |
|---|---|
| 激安 / 割引を煽る表現 | ブランド価値の毀損 |
| 他社を下げる比較表現 | ブランドの品位に反する |
| 「最上級」「No.1」等の優位性断定 | 景表法リスク + 作り手スタンスに反する |
| 不自然な誇大広告 | ブランドの信頼を損なう |
| 過度に個人（代表）を崇拝する表現 | 代表は「現場を知る創業者」として扱う |
| 通販的な煽り構造 | セクションをまたいで文を完成させるなど |

### 6.4 デザインに関する禁止事項

| NG | 理由 |
|---|---|
| MIRANESTのグレー基調色をBI-SUアプリに適用 | BI-SUの基調色はウォームアイボリー × シャンパンゴールド。`#d8d8d8` / `#565656` はMIRANEST専用 |
| 冷たい無機質なグレースケール配色 | BI-SUは温かみのある暖色系が基調 |
| 純白（`#FFFFFF`）を背景のベースカラーにする | アイボリー系（`#F4F1EA` 〜 `#FBFAF6`）を使う |
| AI生成のボタニカル柄 | 自社素材をデザイナーが使用する |
| `vault/Brand/基調色ガイド.md` をBI-SUの色定義として参照する | 基調色ガイドはMIRANESTの定義。BI-SUには本ガイドラインの色を適用 |

### 6.5 動物表現の禁止事項

| NG | 正しい表現 |
|---|---|
| 「愛犬」のみ記載（猫を忘れる） | 「愛犬・愛猫」を併記（施設が犬限定の場合を除く） |
| 飼い主を「飼い主様」「オーナー様」 | 「あなた」（へりくだりすぎない対等な距離感） |

### 6.6 データ表示の禁止事項

| NG | 正しい対応 |
|---|---|
| 推測や仮データを本実装に使用 | モックのダミーデータは本実装に持ち込まない |
| 未確認の制度情報を表示 | OP論点のクローズを待って実装する |
| 税込/税抜を曖昧にする | 常に税込を明示 |

### 6.7 セキュリティに関する禁止事項

| NG | 正しい対応 |
|---|---|
| 会員IDをバーコード以外の用途で画面に平文表示 | マスキング要否はB3で検討 |
| なりすまし対策なしの静的バーコード表示のみ | 動的要素（時計表示等）で補強 |

### 6.8 UI/UXの禁止事項

| NG | 正しい対応 |
|---|---|
| オフラインで会員証バーコードが表示できない | 事前生成 + キャッシュで表示継続（地下店舗対応） |
| エラー時にユーザーを放置 | スケルトン表示 / キャッシュ表示 + 再試行 / オフラインバナー |
| 未ログイン時に全機能をロック | ハイブリッド方式: 基本機能は未ログインでも利用可能 |

## 7. ライティングガイド

### 7.1 基本トーン

| 方針 | 説明 |
|---|---|
| 上品・誠実 | 読んだ人が「これはただの商品ではない」と感じる文章 |
| 力強い・未来志向 | 世界基準・未来を変えるスケール感を持たせる |
| 科学的 | 感覚だけで語らず、根拠を持って届ける |
| 神秘的 | ツバメの巣の天然素材としての奥深さを表現 |
| 自然への敬意 | 人と動物と自然の共生を表現に込める |

### 7.2 UIコピーの書き方

| ルール | 具体例 |
|---|---|
| 見出しは2行構成・助詞止めで余韻 | 上の行で「対象/世界観」、下の行で「誰に/何を」。文末は「〜を。」「〜に。」 |
| セクションは自己完結 | セクション間はテーマの流れでゆるく繋ぐ。文がセクションをまたがない |
| CTAは本編の文脈から自然に繋ぐ | フッター常設で「載せておく」だけは避ける |
| 読み手で解釈が割れる表現は避ける | 「北極星」→「ゴール」、「ペインポイント」→「困りごと」 |
| 受け手起点で書く | 「自分が伝えたい情報」ではなく「相手の頭の中に今何が浮かんでいるか」を起点に |
| コピーは削る | 「こだわり抜いたもの」→「こだわり」。語を足すより削る |

### 7.3 敬称・呼称ルール

| 対象 | 正しい呼称 | NG |
|---|---|---|
| 会員 | 会員 | 会員様 / メンバー / お客様 |
| ランク名 | Member / Silver / Gold / Platinum / Diamond（英語表記） | メンバー（カタカナ） |
| 飼い主 | あなた | 飼い主様 / オーナー様 |
| ペット | 愛犬・愛猫 | ペット（場面による） |
| 会員制度 | CLUB MEMBERS PROGRAM | Members Club / メンバーズクラブ / Member's Club |
| 商品全般 | BI-SU商品 / MIRANEST商品 / 具体名 / 日々のケア | 製品 / プロダクト / 商品（単体） |

### 7.4 数値・単位の表記ルール

| 項目 | 表記ルール | 例 |
|---|---|---|
| 金額 | 円記号 + カンマ区切り + 税込明記 | ¥16,200（税込） |
| 日付 | 年/月/日（曜日）形式 | 2026/06/24（水） |
| ポイント | 数値 + pt | 162pt |
| 還元率 | 100円 = Npt 形式 | 100円 = 1pt |
| 電話番号 | ハイフン区切り | 03-6824-4117 |
| パーセンテージ | 半角数字 + % | 10%OFF |
| 時刻 | 24時間表記 | 10:30〜20:30 |

### 7.5 多言語対応ルール（v2.1）

| ルール | 詳細 |
|---|---|
| 対応言語 | 日本語 / English（2言語） |
