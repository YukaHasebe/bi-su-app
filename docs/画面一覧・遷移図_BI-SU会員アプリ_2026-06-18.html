画面一覧

画面ID 画面名 種別 概要 関連ストーリー
認証・起動フロー
SCR-001 スプラッシュ システム アプリ起動時のBI-SUロゴ表示。フレームワーク標準のスプラッシュ —
SCR-002 ウェルカム（1〜3画面） フルスクリーン 初回起動時のアプリ紹介。会員証・ランク確認・定期管理などの価値を伝える。スキップ可能、2回目以降は非表示 S-1.3
SCR-003 ログイン フルスクリーン メールアドレスまたは電話番号を入力し「認証コードを送信」するフォーム。未登録時はEC会員登録への案内を表示 S-1.1
SCR-004 認証コード入力 フルスクリーン メール/SMSで届いた認証コードを入力。有効期限表示・再送導線あり。認証完了でS3上のEC顧客データと紐付け S-1.1
SCR-005 起動演出 フルスクリーン ランク別ボルネオ・ズーム動画（約5.5秒）。毎回再生・スキップ常設。着地ビジュアル＝ホーム背景。reduced-motion時は静止画 S-2.1
SCR-006 コーチマーク オーバーレイ 初回ログイン後にホーム画面上で表示。QRの出し方・タブの使い方を案内 S-1.3
メインタブ（5タブ・フッター切替）
SCR-100 ホーム タブ 起動演出の着地ビジュアルを背景に、ランク表示・ポイント残高・進捗バー・3Dロゴ・QRボタンを配置。ランク別にカラーテーマ切替 S-3.1, S-3.2, S-3.3, S-3.4
SCR-200 定期情報 タブ 利用中の定期コース一覧（商品画像・名称・金額・次回届け日・ステータス）。解約済みは区別して表示。異常時アラートバッジ S-5.1, S-5.2, S-5.4, S-5.5
SCR-300 購入履歴 タブ 注文一覧（日付・注文番号・商品画像・名称・数量・金額）。定期/通常・ネット/店頭の絞り込み。20件ずつ追加読み込み S-6.1, S-6.2, S-6.3
SCR-400 ストアーズ（リスト） タブ 店舗リスト（外観写真・名称・距離・住所・区分）。位置情報許可時は近い順、未許可時は固定順。リスト⇔マップ切替 S-7.1, S-7.4, S-7.5
SCR-500 設定 タブ メニュー一覧: アナツバメについて/通知設定/登録情報/問い合わせ/利用規約/PP/ログアウト/退会 S-1.4, S-1.5, S-8.1〜S-8.5
サブ画面・詳細
SCR-101 QR表示 モーダル 会員証QRコードを大きく表示。画面輝度を最大に。会員IDテキスト（マスク+タップ開示）。オフラインでもキャッシュから表示 S-4.1, S-4.2, S-4.3, S-4.4
SCR-102 特典一覧 サブ画面 ランク別優待内容の静的一覧（バースデーギフト・シーズナルギフト・イベントご招待等） S-3.1
SCR-201 定期コース詳細 サブ画面 個別コースの詳細（商品・金額・サイクル・ステータス）。変更・スキップ・解約への導線（→ECマイページ）。異常時は対処導線 S-5.2, S-5.3, S-5.5
SCR-301 注文詳細 サブ画面 個別注文の全商品・金額・ステータス・注文番号（コピー可）。経路表示。問い合わせ導線 S-6.4, S-6.5
SCR-401 ストアーズ（マップ） サブ画面 地図上にピン表示。タップで店舗選択→詳細へ。リストとトグル切替 S-7.2
SCR-402 店舗詳細 サブ画面 外観写真・説明文・営業時間・定休日・電話番号・住所・アクセス・フロアマップ。「経路を見る」→地図アプリ連携 S-7.3, S-7.4
SCR-501 アナツバメについて サブ画面 アナツバメ・ボルネオ・巣の採取のブランドストーリー。アプリ内蔵でオフライン閲覧可 S-8.1
SCR-502 通知設定 サブ画面 PUSH通知カテゴリ別ON/OFF（お知らせ/届けリマインド/キャンペーン）。重要なお知らせは常時ON S-8.2
SCR-503 登録情報 サブ画面 メール変更/住所変更/氏名変更/支払い変更の各導線（→ECマイページ） S-8.3
SCR-504 お問い合わせ / FAQ サブ画面 FAQ一覧+窓口案内（フォーム/メール/電話）。受付時間・回答目安 S-8.4
SCR-505 利用規約 サブ画面 利用規約の全文表示 S-8.5
SCR-506 プライバシーポリシー サブ画面 PP全文。位置情報・PUSH通知・利用状況計測の利用目的を記載 S-8.5
SCR-507 アカウント削除 サブ画面 削除の影響説明→確認ボタン→認証コード再入力で実行。完了後はログイン画面へ S-1.5
ダイアログ（OSダイアログ含む）
DLG-001 ログアウト確認 ダイアログ ログアウトの確認。実行→端末データ消去→ログイン画面へ S-1.4
DLG-002 位置情報許可 OSダイアログ ストアーズ初回利用時に用途説明付きで表示。許可しなくてもリスト利用可 S-7.5
DLG-003 PUSH通知許可 OSダイアログ 通知ON操作時に表示。未許可時は端末設定への導線を表示 S-8.2
外部遷移先（アプリ外）
EXT-001 ECマイページ 外部 定期変更/メール変更/住所変更/支払い変更。遷移後にアプリへ戻るとデータ再取得 S-5.3, S-8.3
EXT-002 EC会員登録 外部 未登録のメールアドレス/電話番号でログイン試行時に案内 S-1.1
EXT-003 地図アプリ（OS） 外部 店舗詳細の「経路を見る」からOS地図アプリで経路案内 S-7.3
SECTION 3

遷移フロー一覧

フロー1: 初回起動（未認証）
SCR-001 スプラッシュ → SCR-002 ウェルカム → SCR-003 ログイン → SCR-004 認証コード入力 → SCR-005 起動演出 → SCR-006 コーチマーク → SCR-100 ホーム
フロー2: 通常起動（認証済み）
SCR-001 スプラッシュ → SCR-005 起動演出 → SCR-100 ホーム
フロー3: タブ間遷移
SCR-100 ホーム ↔ SCR-200 定期情報 ↔ SCR-300 購入履歴 ↔ SCR-400 ストアーズ ↔ SCR-500 設定 （フッタータブで任意のタブに直接移動可能）
フロー4: ホーム起点
SCR-100 ホーム → SCR-101 QR表示（モーダル）→ 閉じてホームに戻る
SCR-100 ホーム → SCR-102 特典一覧 → 戻る
フロー5: 定期情報起点
SCR-200 定期情報 → SCR-201 定期コース詳細 → EXT-001 ECマイページ（変更手続き） → 定期情報に戻る（データ再取得）
フロー6: 購入履歴起点
SCR-300 購入履歴 → SCR-301 注文詳細 → SCR-504 お問い合わせ （設定タブのサブ画面へクロス遷移）
フロー7: ストアーズ起点
SCR-400 ストアーズ（リスト）↔ SCR-401 マップ表示 （トグル切替）
SCR-400 / SCR-401 → SCR-402 店舗詳細 → EXT-003 地図アプリ（経路案内）
フロー8: 設定起点
SCR-500 設定 → SCR-501 アナツバメについて → 戻る
SCR-500 設定 → SCR-502 通知設定 → 戻る
SCR-500 設定 → SCR-503 登録情報 → EXT-001 ECマイページ（各変更） → 戻る
SCR-500 設定 → SCR-504 お問い合わせ → 戻る
SCR-500 設定 → SCR-505 利用規約 / SCR-506 PP → 戻る
フロー9: ログアウト / 退会
SCR-500 設定 → DLG-001 ログアウト確認 → SCR-003 ログイン
SCR-500 設定 → SCR-507 アカウント削除 → 認証コード再入力 → SCR-003 ログイン
