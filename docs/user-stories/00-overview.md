## ペルソナ

| ペルソナ | 説明 |
|---|---|
| **会員**（主） | BI-SUの既存会員。ECでの定期購入や店舗での買い物をしている。30〜60代中心で、スマホ操作は得意な人ばかりではない |
| **店舗スタッフ**（副） | 直営店（GINZA SIX・ISETAN SALONE・京都祇園 等）のスタッフ。レジはスマレジを使用 |

※ 本アプリは**既存会員を前提**とする（新規入会導線はv1スコープ外）。

## 確定済みの前提（ストーリー・DoDの土台）

1. **アプリの性格 = 閲覧専用ミラー**。会員情報・定期・履歴の変更操作はECマイページへ誘導して完結する
2. **店舗オペレーション**: 会員証QRをスマレジで読み取り、会員IDを転記する運用
3. **会員ランク**: MEMBER / SILVER / GOLD / PLATINUM / DIAMOND の5段階。当年1/1〜12/31の購入累計で翌年のステータスが確定（毎年再判定）。アプリの表示ランクは**前年確定の公式ステータス**
4. **起動演出**: ランク別のボルネオ・ズーム演出を**毎回再生**（スキップ常設）。演出はランク上下にかかわらず現ステータス準拠
5. ポイント還元率: MEMBER 1% / SILVER 2% / GOLD 3% / PLATINUM 5% / DIAMOND 10%（100円=1〜10pt）

## 共通DoD（全ストーリーに適用）

個別ストーリーのDoDに加え、以下をすべてのストーリーの完了条件とする。

- [ ] BI-SUブランドのトンマナ（アイボリー×シャンパンゴールド・明朝系・上品さ）に沿ったUIである
- [ ] 表示文言がブランド表記ルールに従う（「会員」を使用し「会員様/メンバー」は不使用。ランク名は英字表記）
- [ ] OSの文字サイズ設定（200%まで）でテキストが欠けず、写真背景上のテキストは十分なコントラスト（WCAG AA目安）を確保する
- [ ] 幅320pt級の端末（iPhone SE等）でもレイアウトが崩れない
- [ ] 通信エラー時にアプリが固まらず、再試行またはキャッシュ表示ができる
- [ ] **ECマイページへ遷移する操作は、再ログイン不要または最小限**（※依存: OP-06/OP-24 ID連携方式）で、**アプリへ戻った時点で表示データが再取得される**
- [ ] iOS / Android の両方で同等に動作する

## 記法

- ストーリー: 「**〜として、〜したい。それは〜だからだ**」形式
- DoD: チェックリスト形式。検証可能な表現（数値・観察可能な振る舞い）で書く
- **※依存**: EC側仕様や未決論点（要件定義書のOP-xx）の確定待ち。ストーリー自体の価値は変わらないため削除せず明示する
- **※要決定**: アプリ側で決められる未決定値。仮置き値と候補を明記してある
