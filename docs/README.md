## 画面構成（28画面以上）

### 画面体系

```
BI-SU 会員アプリ
├── 認証・起動（SCR-0xx）
│   ├── SCR-001  スプラッシュ
│   ├── SCR-002  起動演出（ランク別ボルネオ・ズーム）
│   ├── SCR-003  オンボーディング（初回のみ）
│   ├── SCR-010  ログイン
│   └── SCR-011  認証コード入力
│
├── [Tab 1] ホーム（SCR-1xx）
│   ├── SCR-100  ホーム / ステータス画面
│   │   ├── ランク別背景・3Dロゴエンブレム
│   │   ├── ポイント残高・進捗バー・年間購入累計
│   │   └── 店舗ポイント残高（v2.1新設）
│   └── SCR-101  バーコード表示サブ画面
│       └── DLG-101  バーコード拡大モーダル
│
├── [Tab 2] 定期情報（SCR-2xx）
│   ├── SCR-200  定期情報一覧
│   └── SCR-201  定期詳細 → EXT-001（ECマイページ）
│
├── [Tab 3] 購入履歴（SCR-3xx）
│   ├── SCR-300  購入履歴一覧（月別セクション）
│   └── SCR-301  注文詳細
│
├── [Tab 4] ストアーズ（SCR-4xx）
│   ├── SCR-400  店舗一覧（リスト⇔マップ切替）
│   └── SCR-401  店舗詳細 → EXT-003（地図アプリ）
│
├── [Tab 5] 設定（SCR-5xx）
│   ├── SCR-500  設定メニュー（8項目＋アカウント削除）
│   ├── SCR-501  会員情報
│   ├── SCR-502  通知設定
│   ├── SCR-503  お支払い方法
│   ├── SCR-504  お問い合わせ
│   ├── SCR-505  BI-SUについて（ブランドストーリー）
│   ├── SCR-506  利用規約 / プライバシーポリシー
│   ├── SCR-507  アカウント削除（注意事項・再認証）
│   │   └── DLG-507  削除最終確認ダイアログ
│   └── DLG-506  ログアウト確認ダイアログ
│
├── v2.1 追加予定画面（TBD）
│   ├── 店舗ポイント残高・履歴
│   ├── チェックイン
│   ├── 仮会員証
│   └── 多言語設定
│
└── 外部遷移
    ├── EXT-001  ECマイページ
    ├── EXT-002  EC新規登録
    └── EXT-003  地図アプリ
```

### 画面ID命名ルール

| ルール | 説明 |
|---|---|
| 百の位 = フッタータブ番号 | 1=ホーム, 2=定期情報, 3=購入履歴, 4=ストアーズ, 5=設定 |
| x00 | タブのルート画面 |
| x01以降 | サブ画面 |
| 0xx | 認証・起動フロー（タブ到達前） |
| DLG-xxx | ダイアログ / モーダル |
| EXT-xxx | 外部遷移先 |

## 主要システム構成

### ハイブリッド方式（v2.1決定）

| 利用者 | ログイン | 利用可能な機能 |
|---|---|---|
| 未登録ユーザー（非会員） | 不要 | 仮会員証・店舗ポイント・チェックイン・ストアーズ・ブランドコンテンツ・多言語切替・問い合わせ |
| BI-SU会員（EC購入者・店舗来店者） | 必要 | 上記すべて＋定期情報・購入履歴・ランク制度・通知設定・ポイント優遇率 |

### 認証フロー（MIRANEST App準拠）

```
メールアドレス / 電話番号
  → 認証コード（メール / SMS）
    → 認証完了
      → 楽楽EC顧客データと照合（メールアドレス）
        → 一致: データ紐付け
        → 未登録: カスタマークリエイトAPIで顧客新規作成
```

### 会員ランク制度（5段階）

| ランク | 年間購入金額 | ポイント換算率 | バースデーギフト | シーズナルギフト | イベントご招待 |
|---|---|---|---|---|---|
| **MEMBER** | — | 100円 = 1pt | 100pt | - | - |
| SILVER | 30万円〜 | 100円 = 2pt | ○ | - | - |
| GOLD | 50万円〜 | 100円 = 3pt | ○ | ○ | ○ |
| PLATINUM | 100万円〜 | 100円 = 5pt | ○ | ○ | ○ |
| DIAMOND | 200万円〜 | 100円 = 10pt | ○ | ○ | ○ |

- 集計期間: 当年1/1 -- 12/31の購入累計額
- 適用期間: 翌年1/1 -- 12/31（毎年再判定）
- アプリ表示: 前年確定の公式ステータス

| 系統 | 管理元 | ログイン | 備考 |

## ブランドカラー

BI-SUブランドの基調色（bi-su.jp 公式サイト準拠）。詳細は [`07_ブランドガイドライン.md`](07_ブランドガイドライン.md) を参照。

| 用途 | カラー | 備考 |
|---|---|---|
| 背景ベース | ウォームアイボリー `#F4F1EA` 〜 `#FBFAF6` | 温かみのある暖色系 |
| ブランドアクセント | シャンパンゴールド `#C6B484` | BI-SUの象徴色。ナビ・ボタン・装飾 |
| テキスト | インク `#2A2620` | ダークブラウン系 |
| ボタン | ゴールド `#A8895A` 〜 `#BCA474` | プライマリアクション |
| ダークセクション | ニアブラック `#10100E` 〜 `#1C1C1C` | ホーム画面・演出背景 |

**トーン**: 上品・ラグジュアリー（温もりのある高級感）

> **注意**: MIRANESTブランドの基調色（ライトグレー `#d8d8d8` + ミディアムグレー `#565656`）はBI-SUには適用しない。

## 表記ルール

- 「会員」を使用（「会員様」「メンバー」は使わない）
  - 会員ランク階層名としての **Member / Gold / Platinum** 等の英語表記はOK
- 「MIRANEST商品」「BI-SU商品」の表記を使用（「製品」「プロダクト」「商品」単体は使わない）
- **CLUB MEMBERS PROGRAM**（英字表記固定）
- BI-SU に「（美巣）」は付けない
- 愛犬・愛猫で表現（猫を忘れない）
